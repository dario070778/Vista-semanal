# Vista semanal

Aplicación semanal 24 x 7 inspirada en Note24x7.

Funciones principales:

- Semana de lunes a domingo.
- Una única hoja semanal reutilizable.
- Resaltado automático del día y la hora actual.
- Edición tocando una celda.
- Mantener pulsado y arrastrar para repetir o borrar celdas consecutivas del mismo día.
- Colores elegibles por actividad, recordados automáticamente.
- Guardado local automático en el navegador.
